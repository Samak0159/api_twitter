API Twitter


Cliquez sur le logo pour vous connecter à Twitter.

Timeline

Vous aurez la possibilité de visualiser simplement vos derniers tweets.

De plus, il sera également possible de marquer différents tweets comme favoris ou de les supprimer.

Favoris

La fonctionnalité favoris permet de visualiser uniquement vos tweets préférés !

Tweets supprimés

Vos tweets supprimés seront stockés sur une page externe permettant de les récupérés si besoin.

Ceux-ci seront disponible durant une durée déterminée.
